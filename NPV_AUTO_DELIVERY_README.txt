تحویل خودکار NPVTSUB

در این نسخه بعد از تایید کارت‌به‌کارت یا خرید از موجودی، اگر پلن پاسارگارد نامحدود/npv_lock باشد:
1) ربات Device ID ذخیره‌شده را می‌خواند.
2) لینک ساب اختصاصی را بعد از ساخت کاربر پاسارگارد می‌گیرد.
3) تابع npvCreateLockedSubFile اجرا می‌شود.
4) اگر generator روی سرور ست شده باشد، فایل با اسم همان اشتراک ساخته و برای کاربر ارسال می‌شود.
5) اگر generator ست نباشد، ربات به کاربر پیام آماده‌سازی می‌دهد و اطلاعات ساخت دستی را برای ادمین می‌فرستد.

برای فعال شدن ساخت واقعی فایل، دستور سازنده NPVTSUB را داخل:
settings/npvtsub_generator_cmd.txt
قرار بده. متغیرهای قابل استفاده:
{sub} = لینک ساب
{device} = Device ID
{out} = مسیر خروجی فایل
{name} = اسم اشتراک

نمونه:
 /usr/local/bin/npvtsub-maker --sub {sub} --device {device} --out {out} --name {name}
